# DESIGN SYSTEM

## Color Palette
```
Primary Dark:    #1A1A1A (rich black)
Secondary Dark:  #2D2D2D (charcoal)
Accent Gold:     #C9A962 (luxury gold)
Light:           #F5F5F3 (warm white)
Text Light:      #FFFFFF
Text Muted:      #888888
```

## Typography
```
Headlines: "Playfair Display" (elegant serif) - BOLD
Subheadlines: "Cormorant Garamond" (refined serif) - Medium
Body: "Inter" or "Outfit" (clean sans-serif) - Regular
Accent: "Bebas Neue" (for large statements)
```

## Design Rules (What Makes It Look Expensive)
1. WHITESPACE - Let everything breathe (minimum 80px between sections)
2. LARGE TYPE - Headlines at 72-96px on desktop
3. FULL-BLEED IMAGES - Edge to edge, cinematic
4. SUBTLE ANIMATIONS - Fade in on scroll, slight parallax
5. ASYMMETRICAL LAYOUTS - Not everything centered
6. GOLD ACCENTS - Sparingly on buttons, lines, hover states
7. DARK SECTIONS - Alternating dark/light creates drama

# FRAMER TIPS FOR LUXURY LOOK

## Animation Settings
```
Fade In:
- Opacity: 0 → 1
- Y offset: 20px → 0
- Duration: 0.6s
- Ease: ease-out
- Trigger: On scroll into view

Hover on buttons:
- Background: transparent → gold
- Text: gold → black
- Duration: 0.3s
```

## Image Treatment
- Use object-fit: cover for all images
- Add subtle overlay (linear-gradient) to hero images
- Consider adding grain texture overlay for luxury feel

## Spacing Guide
```
Section padding: 100px top/bottom (desktop), 60px (mobile)
Between elements: 40-60px
Between sections: 120px
Max content width: 1200px
```

## Button Styles
```
Primary (gold outline):
- Border: 2px solid #C9A962
- Background: transparent
- Text: #C9A962
- Padding: 16px 32px
- Hover: Background #C9A962, Text #1A1A1A

Secondary (solid gold):
- Background: #C9A962
- Text: #1A1A1A
- Hover: Background #B8994D
```
