```
1. INVISIBLE COMPLEXITY

   - Advanced settings hidden until needed
   - Smart defaults for everything

2. SPEED OVER EVERYTHING
   - App opens in <1 second
   - Search results instant
   - Actions complete immediately
   - No loading screens

3. ONE-HANDED OPERATION
   - Reachable with thumb
   - Voice alternative always available
   - Gestures for common actions

4. BEAUTIFUL BUT NOT DECORATIVE
   - Every pixel has purpose
   - Dark mode default (easier on eyes)
   - Gentle colors, no harsh UI
   - Premium feel
```

## Onboarding (Critical)

**The Problem:** Apps die because onboarding is overwhelming

**The Solution:** Progressive setup

**Day 1:**
- Create account (30 seconds)
- Add ONE bill or ONE family member
- Show the magic ("See? Now it's tracking")
- Don't ask for anything else

**Week 1:**
- Gentle prompts to add more
- "Want to add your car?"
- "Import bills from email?"
- Never mandatory

**Month 1:**
- AI starts recognizing patterns
- Suggestions based on what's added
- "You mentioned a dentist - add them?"

**Key Principle:** The app gets MORE useful over time. Don't front-load work.
